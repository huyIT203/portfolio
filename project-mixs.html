Project Overview

Mixs is a comprehensive full-stack application designed to help users manage their wardrobe, create and save outfits, plan daily OOTD (Outfit Of The Day) using a calendar, and receive smart outfit recommendations based on personal preferences and body characteristics.

The project follows a microservices architecture with three main components: a Node.js/Express backend API for business logic and data management, a Flutter mobile app for cross-platform user experience, and a Python/FastAPI ML service for image processing and machine learning features.

Key technical highlights include implementation of BLoC state management, clean architecture patterns, JWT authentication with access/refresh tokens, and integration with Google and Facebook OAuth for social login.

Clean Architecture

The Flutter app follows clean architecture with clear separation between Presentation (BLoC, Pages, Widgets), Domain (Entities, Use Cases), and Data (Repositories, Data Sources) layers.

Dependency Injection

Uses get_it and injectable for dependency injection, ensuring loose coupling and easy testability of components.

State Management

BLoC pattern with flutter_bloc for predictable, testable state management. Events trigger state changes through business logic.

Containerization

Docker Compose orchestrates all services: PostgreSQL database, Redis cache, Node.js API, and Python ML service for consistent deployment.
