Overview

Firebase is the backend platform I use to build highly scalable mobile applications. Firebase provides many services such as Authentication, Firestore, Storage, and Cloud Functions.

In my projects, I have used Firebase to handle authentication, store real-time data, and implement push notification features.

Services Used

Authentication

  • Email/Password
  • Google Sign-In
  • Phone Authentication
  • Anonymous Auth

Cloud Firestore

  • Document/Collection model
  • Real-time listeners
  • Complex queries
  • Security rules

Cloud Storage

  • File uploads
  • Image compression
  • Download URLs
  • Storage rules

Cloud Messaging

  • Push notifications
  • Topic messaging
  • In-app messaging
  • Analytics integration
