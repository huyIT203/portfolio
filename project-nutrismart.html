Project Overview

NutriSmart is a cross-platform Flutter application designed to help users track and manage their nutrition intelligently. The app provides comprehensive features for monitoring daily food intake, calculating BMR (Basal Metabolic Rate), and analyzing nutritional data over time.

The project includes two main repositories: a Flutter mobile application for the user interface and a Spring Boot Food API backend that provides comprehensive REST endpoints for food management, meal tracking, and user health data.

Key features include Firebase Authentication with social login support (Google, Facebook), real-time data synchronization with Cloud Firestore, local database caching with SQLite, and detailed health analytics with visual charts and reports.

Provider Pattern

Flutter app uses Provider for state management, offering simple and efficient reactive state updates across the widget tree.

Firebase Integration

Real-time data synchronization with Cloud Firestore. Firebase Authentication handles user sessions. Firebase Storage for images.

REST API Backend

Spring Boot 3 provides a robust REST API with JPA/Hibernate for database operations. Comprehensive endpoint documentation via Postman collections.

Local Caching

SQLite database for offline support and local caching. SharedPreferences for user preferences and settings storage.
