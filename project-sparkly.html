Project Overview

Sparkly is a comprehensive full-stack nutrition and health management platform designed to help users track their dietary intake, manage health goals, and discover healthy food options. The platform consists of four main components working together seamlessly.

The ecosystem includes a Flutter mobile application (NutriSmart) for cross-platform user experience, a Spring Boot 3 backend API for business logic and data management, a Python FastAPI nutrition engine for recipe nutrition calculations, and a React admin dashboard for content management.

Key highlights include JWT authentication with Google OAuth2, comprehensive food and ingredient databases with 80+ nutritional indicators, meal diary tracking, health goal management, and a sophisticated nutrition calculation engine with cooking adjustment factors and unit conversions.

Microservices Architecture

The platform follows a microservices pattern with separate services for mobile app, backend API, nutrition calculations, and admin management, ensuring scalability and maintainability.

Provider Pattern

Flutter app uses Provider for state management, offering simple and efficient reactive state updates across the widget tree.

JWT & OAuth2

Secure authentication with JWT tokens and Google OAuth2 integration. Refresh token mechanism for seamless user experience.

Comprehensive Database

PostgreSQL database with 80+ nutritional indicators per ingredient, supporting complex queries and analytics.
